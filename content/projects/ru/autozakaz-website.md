## Обзор проекта

Автозаказ.ру — крупнейший продавец автомобилей из Японии, Кореи и Китая в России. До обращения ко мне у компании был простой устаревший лендинг на Tilda, который не соответствовал масштабу бизнеса и не давал нужных результатов.

## Задача

Создать полноценный корпоративный сайт, который отражает статус лидера рынка. Требования: быстрая загрузка, отличное SEO в высококонкурентной нише, удобный интерфейс для пользователей любого возраста. Функционал — информационные страницы, видеоотзывы клиентов, актуальные контакты менеджеров. И критически важно — защита от DDoS-атак, которым старый сайт регулярно подвергался.

## Решение

Выбрал WordPress — клиент знаком с этой системой и сможет развивать сайт дальше без огромных затрат. Дизайн разработал в Figma на основе нового брендбука, затем реализовал на WordPress. Интегрировал динамические формы Битрикс24 и настроил глубокую аналитику по всем элементам сайта для оценки эффективности.

## Ключевые особенности

- Core Web Vitals >90 на мобильных и десктопе
- SEO-оптимизация для высококонкурентной автомобильной ниши
- Интеграция с Битрикс24 — формы, CRM, аналитика
- Защита инфраструктуры от DDoS-атак
- Детальная аналитика действий пользователей по всему сайту
- Чистый современный дизайн, понятный пользователям любого возраста

## Результаты

- Вместо простого лендинга — полноценный корпоративный сайт лидера рынка
- Конверсия Битрикс-форм выросла на 300%
- Сайт запущен за 2 месяца
- Сразу после запуска — рост позиций в Яндексе и Google
