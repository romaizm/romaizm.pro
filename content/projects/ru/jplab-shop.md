## Обзор проекта

JPLAB-SHOP продаёт косметику из стран Азии по всей России. Компании нужен был не просто магазин, а полноценная платформа с продвинутыми функциями для удобства покупателей.

## Задача

Создать современный интернет-магазин с расширенным функционалом: умный поиск, личный кабинет с вишлистом, интеграция с популярными службами доставки. Плюс — система видеоотзывов с поддержкой разных платформ.

## Решение

Построил магазин на WordPress + WooCommerce с использованием Breakdance для вёрстки. Написал кастомные PHP-модули для специфических функций: живой поиск, интеграция видеоотзывов с ВКонтакте, RuTube и YouTube, «умное» подключение к API служб доставки.

## Ключевые особенности

- Интерактивный живой поиск — результаты появляются по мере ввода
- Личный кабинет с вишлистом и отслеживанием заказов
- Интеграция доставки со СДЭК и Почтой России с автоматическим расчётом
- Чёткая структура каталога — легко найти нужный товар
- Система видеоотзывов с поддержкой ВКонтакте, RuTube и YouTube

## Результаты

- Покупатели получили удобный инструмент для выбора косметики
- Автоматический расчёт доставки снизил нагрузку на менеджеров
- Видеоотзывы повышают доверие и помогают с выбором товара
