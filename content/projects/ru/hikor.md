## Обзор проекта

Hikor — компания по импорту автомобилей из Южной Кореи. Задача — создать сайт для оптового направления бизнеса, ориентированного на работу с юридическими лицами.

## Задача

Разработать презентационный сайт для B2B-сегмента, который производит впечатление на потенциальных партнёров и выделяется на фоне конкурентов. Сайт должен быть визуально эффектным и демонстрировать серьёзность компании.

## Решение

Создал сайт на WordPress с уникальным дизайном, разработанным в Figma. Особое внимание уделил визуальной части — качественные изображения, продвинутые анимации, плавные переходы. Всё это создаёт ощущение премиальности, что важно для B2B-сегмента.

## Ключевые особенности

- Уникальный дизайн с фокусом на B2B-аудиторию
- Продвинутые анимации и плавные переходы
- Премиальный визуал, соответствующий позиционированию компании
- Адаптивная вёрстка для всех устройств

## Результаты

- Компания получила презентационный сайт для работы с оптовыми клиентами
- Визуально выделяется среди конкурентов в нише автоимпорта
