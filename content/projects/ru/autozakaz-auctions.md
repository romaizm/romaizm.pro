## Обзор проекта

Для компании «Автозаказ» я разработал современную систему поиска автомобилей на крупнейших аукционах Азии — Японии, Кореи и Китая. Задача — дать клиентам компании удобный инструмент для самостоятельного подбора авто, который работает быстро и выглядит понятно.

## Задача

На рынке есть готовые решения для работы с аукционами, но они негибкие, непрозрачные и дорогие. Клиенту нужна была собственная система с нуля: удобный интерфейс, высокая скорость, возможность развивать продукт дальше. И всё это — за 3 месяца.

## Решение

Подключился к API поставщиков аукционных каталогов и построил поверх них собственное веб-приложение на React. Интерфейс спроектировал так, чтобы спрятать всю техническую сложность — пользователь видит только понятный путь от поиска до детального изучения автомобиля. Визуал выдержан в фирменном стиле бренда «Автозаказ».

## Ключевые особенности

- Кастомный функционал под специфику бизнеса клиента
- Точные расчёты стоимости с учётом всех нюансов импорта
- Быстрая работа даже с огромными каталогами
- Чистый интерфейс, который не перегружает пользователя
- Адаптивная вёрстка для любых устройств

## Результаты

- Клиенты компании получили удобный инструмент для самостоятельного поиска авто
- Сложные аукционные каталоги превратились в понятный и логичный интерфейс
- От концепции до релиза — 3 месяца
