## Обзор проекта

Yuwab — стартап из ОАЭ с амбициозной идеей: сделать поиск профессионального коуча таким же простым, как свайп в Tinder. Бизнес-коучи, личные, семейные — все в одном приложении.

Пользователь проходит короткий опрос о себе, а затем листает карточки коучей: свайп вправо — нравится, влево — не подходит. После матча можно начать общение и записаться на сессию.

Проект я вёл с нуля до готового продукта, после чего передал приложение крупному подрядчику для публикации в AppStore и Google Play и дальнейшей поддержки.

## Задача

Стартовая точка — идея на бумаге: «Tinder для поиска коучей». Из этого нужно было создать полноценный продукт.

Ключевые требования: одно приложение для iOS и Android (чтобы не платить за разработку дважды), игривый и понятный интерфейс, подписочная модель оплаты через Apple и Google, встроенный чат с коучами и интеграция с Zoom для видеосессий.

## Решение

Выбрал Flutter — фреймворк от Google, который позволяет из одного кода собрать приложение сразу под обе платформы. Это сэкономило клиенту время и бюджет.

Дизайн разработал на основе брендбука с учётом особенностей мобильных интерфейсов. Под капотом — проверенный стек: Firebase для базы данных, авторизации и пуш-уведомлений, RevenueCat для управления подписками, Sentry для отслеживания ошибок, CodeMagic для автоматической сборки. Бэк-офис клиента работает на Odoo — туда автоматически попадают данные о пользователях и платежах.

## Ключевые особенности

- Единое приложение для iOS и Android с идентичным дизайном
- Двуязычный интерфейс: английский и арабский
- Умный опросник, который подбирает коучей под запросы пользователя
- Подписки через встроенные покупки Apple и Google
- Чат с коучами прямо в приложении
- Видеосессии через Zoom с AI-саммари встречи
- Интеграция с ERP-системой Odoo для управления бизнесом
