## Project Overview

VladCar125 is a car import company. They needed a simple, clear website to present the company and build trust with potential customers.

## Challenge

Create a landing page with clean design and the ability to publish customer reviews. The company had no brand identity, so a basic brand book also needed to be developed.

## Solution

Built a WordPress landing page with a custom design for the client. Also prepared a simple brand book — colors, fonts, basic elements. The site was optimized for loading speed.

## Key Features

- Clean design created from scratch
- Ability to add and manage customer reviews
- Fast loading — Core Web Vitals 90+
- Basic brand book included

## Results

- Company received a modern website and brand identity foundation
- Core Web Vitals 90+ — site runs fast
- Easy content and review management
